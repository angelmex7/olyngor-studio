# Olyngor Studio Website — Production Package

Complete static website prepared for GitHub Pages.

## Included

- `index.html` — Home page.
- `about.html` — About page.
- `assets/css/styles.css` — Native CSS.
- `assets/images/logo-completo.svg` — Full desktop logo.
- `assets/images/logo-icono.svg` — Mobile/icon logo.
- `assets/images/favicon.svg` — SVG favicon.
- `assets/images/contact-email.png` — Contact email displayed as an image to reduce scraping and spam.
- `docs/Legal.pdf` — Legal information.
- `docs/privacy-olyngor.pdf` — Privacy information.
- `robots.txt` — Basic crawler instructions.
- `sitemap.xml` — Basic sitemap.

## Final implementation notes

- Semantic HTML5 and native CSS.
- No JavaScript required.
- No analytics, trackers, cookies, or tracking pixels included.
- Legal and Privacy open as PDF files in a new tab using `target="_blank"` and `rel="noopener"`.
- Contact email is not written as plain text and does not use a `mailto:` link.
- Contact email is displayed as transparent PNG using corporate text color `#2D3748`.

## Publishing on GitHub Pages

Upload all files to the repository root and enable GitHub Pages from Settings → Pages.

## Domain note

When the final domain is available, replace `https://example.com` in `robots.txt` and `sitemap.xml`.
